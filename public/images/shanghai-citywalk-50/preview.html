重要提醒：本文仅用于患者教育与科普交流，帮助患者和家属理解公开研究信息、提高与医生沟通的效率。具体诊断、分期判断、用药选择、剂量调整、联合方案和不良反应处理，必须由具备资质的临床医生结合患者完整病情后决定。

很多消化道肿瘤患者在病理报告或基因检测报告中，会看到 ERBB2扩增、HER2 IHC 2+/3+、FISH阳性、ISH阳性、HER2高表达 等字样。它们提示肿瘤可能存在HER2通路异常，也意味着在部分癌种和特定治疗线数中，患者可能有机会评估HER2靶向治疗。

但HER2治疗并不是“看到一个阳性就可以直接使用某个药”。不同药物对应的癌种、治疗线数、HER2检测标准、获批适应症和可及性不同；不同研究之间也不能简单横向比较谁一定更强。

本文整理4个常被讨论的HER2方向药物：SHR-A1811 / 瑞康曲妥珠单抗、德曲妥珠单抗 T-DXd / DS-8201、曲妥珠单抗、泽尼达妥单抗 Zanidatamab。重点帮助患者理解后线治疗参考信息，并与主治医生讨论是否适合HER2 ADC、HER2双抗、曲妥珠单抗联合化疗或其他靶化组合方案。

先看清：HER2阳性不是一个单一概念

📋 问诊前建议先核对报告中的4个信息

1. HER2 IHC：是0、1+、2+还是3+？多数明确获益人群集中在IHC3+或IHC2+/ISH阳性。

2. ISH/FISH：是否提示ERBB2扩增或HER2基因扩增？如果只是普通IHC2+，不能自动等同于HER2阳性。

3. 癌种与部位：胃/胃食管结合部癌、结直肠癌、胆道癌、胰腺癌等证据强弱不同。

4. 治疗线数和既往用药：一线、二线、三线后的方案选择差异很大，是否接受过曲妥珠单抗也会影响判断。

四种HER2药物：患者版详细对比

A · HER2 ADC SHR-A1811 / 瑞康曲妥珠单抗

Trastuzumab rezetecan｜恒瑞医药｜消化道适应症仍在推进

药物类型：SHR-A1811是一种HER2靶向抗体偶联药物（ADC）。简单理解，它把识别HER2的抗体、可切割连接子和拓扑异构酶I抑制剂载荷连接在一起，让药物更集中地进入HER2表达的肿瘤细胞内释放毒素。由于载荷具有一定膜透过性，也可能产生“旁观者效应”，对邻近肿瘤细胞产生杀伤。

胃癌 / 胃食管结合部癌数据：在HER2表达晚期GC/GEJ与CRC的多中心开放标签I期研究中，HER2阳性GC/GEJ患者n=40，ORR为45.0%，DCR为87.5%，DoR为5.5个月，mPFS为9.0个月，mOS为16.3个月。在HER2 IHC2+/ISH-的GC/GEJ患者n=12中，ORR为25.0%，DCR为91.7%，mPFS为12.2个月，mOS尚未成熟。DCR等部分具体数值建议在信源中注明为恒瑞官方新闻引述JCO 2026研究数据。

结直肠癌数据：HER2阳性结直肠癌患者n=37，ORR为40.5%，DCR为91.9%，DoR为6.1个月，mPFS为9.5个月，mOS为22.7个月。

胆道癌数据：HORIZON-X公开信息显示，在HER2阳性非乳腺癌中的胆道癌BTC亚组，ORR为38.3%，mPFS为7.9个月。这里需要特别注意：不应再写成mPFS 5.5个月。

毒性重点：HORIZON-X中，≥3级治疗相关不良事件为65.9%；需要重点关注血小板下降、贫血、中性粒细胞下降等骨髓抑制相关问题；ILD/间质性肺病为2.5%。对于血小板低、既往出血、骨髓储备差、肺部基础疾病患者，需要医生特别评估风险。

上市/进展：SHR-A1811在中国已获批的适应症为HER2突变NSCLC，非消化道适应症。消化道方向仍在推进中：结直肠癌新适应症上市申请已受理并纳入优先审评；胃/胃食管结合部癌二线III期研究进行中。对于ERBB2扩增或HER2高表达患者，现实意义是可与医生讨论临床试验、可及性或未来用药机会。

B · HER2 ADC 德曲妥珠单抗 / T-DXd / DS-8201

Fam-trastuzumab deruxtecan-nxki｜Enhertu｜证据链较成熟

药物类型：德曲妥珠单抗是一种HER2 ADC，由HER2抗体、连接子和DXd拓扑异构酶I抑制剂载荷组成。它是目前HER2 ADC中证据链较完整、国际批准较广的药物之一。与普通单抗相比，ADC不只是“堵信号”，还承担了将细胞毒载荷递送到肿瘤细胞内的作用。

胃癌 / 胃食管结合部癌数据：FDA 2021年批准T-DXd用于既往接受过曲妥珠单抗方案的HER2阳性局晚或转移性胃/GEJ腺癌。DESTINY-Gastric01中，T-DXd组mOS为12.5个月，对照组为8.4个月；确认ORR为40.5%，对照组为11.3%。

结直肠癌数据：DESTINY-CRC01最终结果显示，在HER2阳性mCRC cohort A（IHC3+或IHC2+/ISH+）中，ORR为45.3%，DCR为83.0%，mPFS为6.9个月，mOS为15.5个月。值得注意的是，HER2低表达队列（IHC2+/ISH-或IHC1+）未观察到明确缓解。

毒性重点：T-DXd最需要警惕的是ILD/肺炎风险，说明书中有相关黑框警告。患者一旦出现新发或加重的咳嗽、气促、发热、胸闷、影像学磨玻璃影等，需要及时告知医生。其他常见不良反应包括贫血、白细胞下降、中性粒细胞下降、淋巴细胞下降、血小板下降、恶心、食欲下降、疲乏、肝酶升高、腹泻等。

上市/进展：美国2021年批准T-DXd用于HER2阳性胃/GEJ后线；2024年FDA加速批准用于既往治疗后、无满意替代方案的HER2阳性IHC3+泛实体瘤。中国胃癌/胃食管结合部癌相关适应症和具体可及性，请以国家药监局最新批准信息、药品说明书以及医院药事管理为准。

C · HER2单抗 曲妥珠单抗 / Trastuzumab

Herceptin｜HER2阳性胃/GEJ一线治疗基础药物

药物类型：曲妥珠单抗是一种HER2单克隆抗体，不是ADC，本身不携带化疗毒素。它通过结合HER2受体，抑制HER2相关信号通路，并可参与抗体依赖性细胞毒作用。临床上在消化道肿瘤中最经典的定位，是HER2阳性转移性胃癌/胃食管结合部癌一线联合化疗。

胃癌 / 胃食管结合部癌数据：ToGA研究奠定了HER2阳性转移性胃癌/GEJ一线“曲妥珠单抗 + 化疗”的基础。FDA Herceptin说明书中，ToGA更新OS显示：曲妥珠单抗+氟尿嘧啶类/顺铂化疗组为13.1个月，对照化疗组为11.7个月，HR为0.80。在HER2高表达亚组（IHC3+ / FISH±）中，OS为18.0个月 vs 13.2个月，HR为0.66。

在后线中的理解：曲妥珠单抗不是后线最强的HER2药物，但它是HER2治疗体系的“基础骨架”。部分患者在后线可能会讨论曲妥珠单抗再挑战、曲妥珠单抗联合化疗、联合其他HER2靶向或抗血管药物等策略，但这需要结合既往治疗反应、HER2状态是否仍然阳性、疾病进展速度和患者耐受性。

毒性重点：曲妥珠单抗核心风险是心肌病、LVEF下降、输液反应和肺毒性。与化疗联用时，还会叠加化疗相关毒性，例如中性粒细胞下降、腹泻、疲乏、贫血、口腔炎、体重下降、血小板下降等。因此，治疗前和治疗过程中通常需要医生评估心功能，并根据患者体力状态、血象、肝肾功能决定联合方案。

上市/进展：曲妥珠单抗已是HER2过表达转移性胃/GEJ腺癌一线联合顺铂和卡培他滨或5-FU的经典方案之一，可及性相对较好。但对于已经多线治疗后的患者，是否继续围绕曲妥珠单抗做联合，需要与医生讨论其证据强度和实际获益可能。

D · HER2双抗 泽尼达妥单抗 / Zanidatamab

Ziihera｜HER2双特异性/双表位抗体｜非ADC

药物类型：泽尼达妥单抗是一种HER2双特异性/双表位抗体，非ADC。它不是携带化疗毒素的药物，而是通过同时结合HER2蛋白上的两个不同表位，增强HER2受体聚集、内化和免疫效应。它的毒性谱与ADC不同，不能简单用ADC的风险框架来理解。

胆道癌获批依据：FDA 2024年加速批准zanidatamab-hrii用于既往治疗后、不可切除或转移性HER2阳性IHC3+胆道癌。FDA批准依据人群为IHC3+ BTC患者n=62，ORR为52%，mDOR为14.9个月。

最终分析数据：JAMA Oncology 2026最终分析显示，HERIZON-BTC-01纳入IHC2+或IHC3+且具有HER2扩增/ISH阳性背景的患者共80例，总体cORR为41.3%，mDOR为14.9个月，mOS为15.5个月。其中IHC3+队列n=62，cORR为51.6%，mPFS为7.2个月，mOS为18.1个月；IHC2+队列n=18，仅1例部分缓解，cORR为5.6%，mPFS为1.7个月，mOS为5.2个月。

入组表述要特别严谨：HERIZON-BTC-01不能写成“普通IHC2+患者也明显获益”。更准确的写法是：HER2阳性BTC入组为IHC2+或IHC3+，且具有ISH/FISH阳性或HER2扩增背景；FDA获批核心疗效人群为IHC3+。如果患者报告只是IHC2+，必须进一步确认ISH/FISH是否阳性。

毒性重点：泽尼达妥常见不良反应包括腹泻、输液相关反应、腹痛和乏力；需要关注LVEF下降、严重腹泻和输液反应。最终分析中，治疗相关不良事件发生率为76%；较常见的3级治疗相关不良事件包括腹泻、射血分数下降和贫血；未报告治疗相关死亡。与ADC不同，骨髓抑制和ILD不是最突出的毒性重点。

上市/进展：泽尼达妥在美国于2024年加速批准，用于既往治疗后的HER2阳性IHC3+不可切除或转移性胆道癌。对于HER2 IHC3+或ERBB2扩增胆道癌患者，尤其值得在后线咨询主治医生；其他消化道瘤种仍以研究探索为主。

拿着报告，可以这样和医生沟通

1. 我的HER2结果到底是 IHC3+、IHC2+/ISH+、FISH阳性，还是ERBB2扩增？是否需要复核病理或重新检测进展病灶？

2. 我现在属于第几线治疗？既往是否用过曲妥珠单抗？是否适合评估 HER2 ADC、HER2双抗、曲妥珠单抗联合化疗或其他靶化组合？

3. 如果考虑ADC类药物，我的肺部基础情况、既往肺炎/间质性肺病、血小板、白细胞、中性粒细胞、贫血情况是否允许？

4. 如果考虑曲妥珠单抗或泽尼达妥，是否需要先做心脏彩超或LVEF评估？是否存在输液反应、严重腹泻或心功能下降风险？

5. 当前当地是否可及？是否有临床试验、注册研究、院内可及、慈善援助或其他合规路径？

为什么不能直接横向比较？
因为这些研究的癌种、入组标准、HER2检测方式、治疗线数、既往用药、样本量、随访时间和终点设计都不一样。例如，泽尼达妥的强证据主要集中在HER2阳性IHC3+胆道癌；曲妥珠单抗主要是胃/GEJ一线联合化疗基础；T-DXd有更成熟的多癌种批准；SHR-A1811则在多个消化道瘤种中显示积极早期信号，但适应症仍在推进中。因此，本表更适合用于理解“可问医生的问题”，而不是自行决定用药排序。
